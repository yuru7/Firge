Fira Mono と源真ゴシックを合成したプログラミングフォント **Firge (ファージ)**

ダウンロードはこちら: [Releases](https://github.com/yuru7/Firge/releases)

- Firge: 文字幅が全角1文字:半角2文字になっている、クセの少ないフォントファミリー
- Firge35: Firge の文字幅を全角3文字:半角5文字に変更したフォントファミリー
